Requirements

FR-01.01 User Authentication Done

The system SHALL provide user authentication via email/password and OAuth 2.0 (Google). Sessions expire after 24 hours by default. A "remember me" option extends persistence to 30 days.

FR-01.02 Dashboard Overview In Progress

The system SHALL display a dashboard with the following widgets:

  • Active project count and status summary
  • Recent activity feed (last 20 events)
  • Pipeline health indicator per project
  • Quick-action buttons for common tasks

FR-01.03 REST API To Do

The system SHALL expose a RESTful API following OpenAPI 3.1 conventions. All endpoints require authentication via Bearer token. Rate limiting applies at 100 requests/minute per user.

NFR-01 Performance To Do

The system SHALL respond to API requests within 200ms (p95) under normal load. Dashboard page load SHALL NOT exceed 1.5 seconds on a 4G connection.
