## Abstract

Shipwright is an AI-powered SDLC framework built on Claude Code. It is structured as a monorepo of Claude Code plugins (one per SDLC phase) with shared scripts, profiles, and templates. End users invoke individual phases (project, plan, build, test, security, deploy, changelog, compliance, iterate, preview, design, adopt) via slash commands; the run plugin orchestrates them end-to-end. The repo itself is a library/framework — there is no deployable web service in the repo root; the optional Command Center WebUI lives in a separate repository.
