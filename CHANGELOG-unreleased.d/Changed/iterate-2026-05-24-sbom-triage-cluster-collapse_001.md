SBOM triage producer collapses N≥2 workspaces sharing the same undeclared-dep signature into one cluster action-unit (`sbom:undeclared-cluster:<sha256-12>`) instead of N per-workspace items. Cluster dedup key encodes both signature AND member-list so growing/shrinking the cluster auto-dismisses the old key and emits a new one. Per-workspace shape preserved for N=1. Shadow per-workspace keys in `current_keys` shield legacy items from accidental dismiss when a workspace joins a cluster.
