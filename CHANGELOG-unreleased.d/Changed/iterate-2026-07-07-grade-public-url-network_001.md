shipwright-grade: a public URL / `owner/repo` target now defaults to GitHub network enrichment (reviewed-PR provenance, CI test tiers, code-scanning SARIF) — its identity is already sent to clone it, so there is no privacy reason to stay local. A **local path** stays local-only by default (privacy-first, even with a public remote); a private/unverifiable remote auto-disables enrichment unless `--allow-network-private`; and an unavailable `gh` falls back to the honest local grade, never a false F.
