shipwright-grade now lights up more Control-Grade dimensions on a cold repo: maintainability (static oversize-file ratio) and dependency hygiene (lockfile licenses) locally, plus test-health (best-available CI JUnit / OpenSSF-Scorecard check-runs / static inventory) and security (GitHub code-scanning) with the new opt-in `--allow-network` flag. Default stays local-only so private repos never leave the machine; a private remote auto-disables enrichment unless `--allow-network-private` is passed.
