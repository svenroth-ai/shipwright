**Cross-repo output contracts.** `/shipwright-grade`'s report (`grade.py --format json`) and `/shipwright-adopt`'s `.shipwright/adopt/snapshot.json` now carry a `schema_version` (`major.minor`), so a consumer can detect a shape it does not understand and say so instead of rendering nonsense. Both are rendered field-for-field by the Command Center WebUI; each plugin's SKILL.md now states the contract and names the consumer. A contract gate per producer diffs the emitted payload against the fixture published on `origin/main` — a baseline a pull request cannot rewrite — derives the bump that diff obliges (breaking → major, additive → minor) and fails until it has been performed.
