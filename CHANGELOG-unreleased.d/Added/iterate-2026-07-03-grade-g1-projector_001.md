shipwright-grade (campaign G1): a read-only /shipwright-grade plugin that grades any LOCAL git repository against the Control Grade rubric (A-F), reusing the compliance scoring engine UNCHANGED. It projects git history, tests and configs into GradeInputs; underivable dimensions render honest n/a and surface as 'controls Shipwright would light up'. Deterministic terminal/markdown/json card; local-only, untrusted-input hardened (ANSI/bidi strip, list-arg git, bounded caps).
