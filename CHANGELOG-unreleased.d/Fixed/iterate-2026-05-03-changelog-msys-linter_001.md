**changelog:** `aggregate_changelog.py` now lints drop-file bullets for Git-Bash MSYS path-mangling artifacts (the `C:/Program Files/Git/...` prefix Bash on Windows injects when an argv arg starts with a slash). Default behavior is a stderr WARN per finding; new `--strict` flag elevates to a hard fail (recommended for release CI). `write_changelog_drop.py` also emits a defensive WARN at write time. Caught at v0.15.0 release prep — closes ADR-023.
