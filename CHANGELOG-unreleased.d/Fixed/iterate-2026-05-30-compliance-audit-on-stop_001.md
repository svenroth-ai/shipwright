Compliance triage items (`source=compliance`) now auto-emit and auto-dismiss on every iterate/changelog Stop via the new `audit_compliance_on_stop.py` hook, instead of only when `/shipwright-compliance` is run manually. A full-coverage safety gate runs the entire A-G audit and refuses to mirror on any partial/crashed run, so a missing group can never wrongly dismiss another group's items. Idempotent per (HEAD-sha, session); opt-out via `SHIPWRIGHT_COMPLIANCE_AUDIT_ON_STOP=0`.
