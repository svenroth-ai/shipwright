Detective-audit reports no longer leak as untracked files in adopted repos: `run_audit.py` now writes the JSON payload to `.shipwright/compliance/audit-report.json` (relocated from the repo root) and the gitignore canon re-excludes both `audit-report.{md,json}` — propagating the ignore to every adopted repo (the root path previously escaped the `.shipwright/`-scoped canon). stdout stays the stable JSON contract. Completes the ADR-089 producer-leak class started by idle-main-artifact-hygiene.
