# Mini Todo App

Build a minimal todo list application with Supabase and Next.js.

## Features

- Users can create, read, update, and delete todo items
- Each todo has: title, description, status (open/done), due date
- Simple authentication (email/password)
- Dashboard showing all todos with filter by status

## Constraints

- Must use Supabase for backend (auth + database)
- Must use Next.js App Router
- Must use Tailwind CSS for styling
