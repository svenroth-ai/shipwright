# Untitled UI Component Reference

> https://www.untitledui.com/react/components

## Overview

Untitled UI is a React component library with a design system. For mockups,
we translate its visual patterns into standalone HTML+CSS.

## Key Components to Replicate in Mockups

### Navigation
- **Sidebar Navigation**: Vertical nav with icons + labels, collapsible
- **Top Navigation**: Horizontal nav bar with logo, links, user menu
- **Breadcrumbs**: Path indicator for nested pages
- **Tabs**: Horizontal tab bar for in-page navigation

### Data Display
- **Table**: Sortable columns, row actions, bulk select
- **Card**: Content container with header, body, footer
- **Stats/Metrics**: Number + label + trend indicator
- **Avatar**: User image with fallback initials
- **Badge**: Status indicator (success, warning, error, info)

### Input
- **Text Input**: Label + input + helper text + error state
- **Select/Dropdown**: Single and multi-select
- **Checkbox / Radio**: With labels and groups
- **Toggle Switch**: On/off with label
- **Date Picker**: Calendar dropdown
- **File Upload**: Drag-and-drop zone

### Feedback
- **Toast/Notification**: Top-right, auto-dismiss
- **Alert**: Inline message with icon
- **Modal/Dialog**: Overlay with actions
- **Progress Bar**: Linear or circular
- **Skeleton Loader**: Content placeholder animation

### Layout
- **Page Header**: Title + description + actions
- **Section Divider**: Horizontal rule with optional label
- **Empty State**: Illustration + text + CTA

## Mockup Translation Rules

When generating HTML mockups:
1. Use CSS custom properties for theming (easy to change colors)
2. Replicate Untitled UI's spacing (8px base grid)
3. Use Inter font (matches Untitled UI default)
4. Include hover/focus states in CSS
5. Use realistic data, not placeholders
